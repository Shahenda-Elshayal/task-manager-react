# Smart Task Manager

A sleek and intuitive to-do list application built with React and Material-UI, designed to help users manage their tasks efficiently.

## Features

- **Add Tasks**: Create tasks with a title and optional details.
- **Mark as Complete**: Toggle task completion status with a single click.
- **Edit Tasks**: Update task details with ease.
- **Delete Tasks**: Remove tasks permanently.
- **Persistent Storage**: Tasks are saved in `localStorage` for continuity.
- **Responsive Design**: Optimized for devices of all sizes.
